Roles Â¿Para que sirven los roles?

Pues estos son lo que hace el tema de los permisos. Cada rol es configurable para que los usuarios tengan permiso a hacer ciertas cosas, y tengan prohibido hacer otras. Un ejemplo de esto podria ser el rol de moderador, este normalmente modera los canales para revisar que los usuarios cumplan con las reglas del servidor, y suele tener permisos como:

Banear usuarios.
Esto depende del servidor, pero consiste en hacer que el usuario sea expulsado, si este baneo es permanente el usuario no tendra forma de volve en esa cuenta.
Aislar temporalmente a un usuario.
Hace que el usuario no tenga forma de comunicarse durante un tiempo, el tiempo es elegido por la persona que pone el aislamiento. Puede ser de 1 minuto hasta 1 semana completa. Esto dependera de la infracciÃ³n del usuario aislado.
Eliminar mensajes.
Normalmente un usuario normal solo es capaz de eliminar sus propios mensajes, pero normalmente un moderador puede eliminar mensajes de cualquier persona, excepto a los que tienen un rol mas alto que el suyo o mas importante.
Usar comandos especiales.
Si el servidor cuenta con bots, puede usar comandos que requieren permisos especiales, como cerrar un canal de texto, banear directamente con el bot, etc.
